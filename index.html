Knowledge On

Web Design

Web design refers to the design of websites that are displayed on the internet. It usually refers to the user experience aspects of website development rather than software development.

Web Development

Web development is the process of building websites and applications for the internet, or for a private network known as an intranet. Web development is not concerned with the design of a website; rather, it's all about the coding and programming that powers the website's functionality

Software Development

Software development is the process programmers use to build computer programs. The process, also known as the Software Development Life Cycle (SDLC), includes several phases that provide a method for building products that meet technical specifications and user requirements.
